# AI Powered Customer Support Intelligence System

An AI-based system that converts customer messages into structured support tickets using NLP.

## Features
- Multi-label issue detection
- Priority classification
- Ticket logging system
- Admin analytics dashboard
- Explainable AI (keywords)

## Tech Stack
- Python
- Streamlit
- Machine Learning (NLP)

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
